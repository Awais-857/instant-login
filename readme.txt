== Description ==
Creates a custom login page with AJAX validation upon activation. Features include:

- Auto-generated login page at /login
- Smooth AJAX form submission with loading spinner
- Customizable success/error messages
- Optional URL redirection after successful login
- Clean, responsive design

== Installation ==
1. Upload the `instant-login` folder to `/wp-content/plugins/`
2. Activate the plugin through WordPress admin
3. Visit `yoursite.com/login` to access your new login page

== Configuration ==

2. Set a redirect URL (optional)
   - Leave blank to show success messages
   - Enter URL to redirect after successful login
